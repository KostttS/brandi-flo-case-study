# Brandi Flo — E-commerce Mobile App

Mobile e-commerce app for branded products with catalog, filters, wishlist, order history, payment flow, push notifications and app store release preparation.

The app was built for a real commercial project and focused on a complete shopping experience for mobile users.

## Technical challenges

### Payment integration

The app required an external payment flow with a return back to the mobile app after payment.

I implemented the payment scenario, deep link return logic and payment status checking flow.

### Store release preparation

The project required preparing the app for production release, including app metadata, store requirements, payment flow clarification and mobile build checks.

### Firebase-based mobile backend

The app uses Firebase services for authentication, database, storage, push notifications and production app logic.
